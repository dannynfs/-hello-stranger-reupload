此作品為大四畢專作品 
"Hello World" 藍芽室內定位館藏導覽裝置與系統



系統除硬體裝置還包含管理系統及使用者APP
持APP靠近任一裝置即可觸發對應裝置的音頻導覽



鄙人於此作品中負責所有硬體相關
包含自訂藍芽封包，與主伺服器的網路接應/資料傳送，及設計一套能偵測電池電量的電路系統



此次作品花最久時間突破的地方是電量偵測的部分
由於非本科專業，鄙人參閱許多網路資料且諮詢其他科系的學生
終於將電壓過大燒壞的問題解決
